## student performance dashboard 

# Project Overview
This Power BI dashboard provides an analytical view of student academic performance. It is designed to help educators monitor educational outcomes, identify trends in student achievement, and detect individuals who may require additional academic support.

# Key Metrics and KPIs
Average Score: The mean grade calculated across all students and subjects.

Pass Rate: The percentage of the student body that has met or exceeded the passing threshold.

Enrollment Count: Total number of active students recorded in the dataset.

Attendance Correlation: An analysis of how class attendance impact final exam results.

# Visualization Breakdown
Subject Performance: A comparison of average marks across different departments such as Math, Science, and English.

Grade Distribution: A visual spread showing the volume of students falling into specific grade brackets (e.g., A, B, C, D, F).

Geographic/Demographic Distribution: Analysis of performance based on student location or other demographic segments.

Performance Trends: A time-series analysis showing how scores have evolved over different semesters or exam cycles.

# Data Structure
The report utilizes data from the following fields:

Student Metadata: Name, ID, Gender, and Region.

Academic Records: Subject names, Score values, and Exam dates.

Logistics: Attendance records and shipping/resource modes (if applicable to the specific dataset).
